# Wiki

## Обзор

Внутренняя Wiki-система для документации проекта, обмена знаниями и коллаборации команды.

## Структура

- `api/`: API для управления страницами и контентом.
- `services/`: Логика управления версиями и рендеринга Markdown.
- `domain/`: Модели страниц и разделов.

## Основные возможности

- Создание и редактирование страниц (Markdown).
- История изменений.
- Поиск по контенту.
