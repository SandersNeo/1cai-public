# Admin Dashboard

## Обзор

Модуль административной панели для управления платформой 1C AI Stack. Предоставляет инструменты для просмотра глобальной статистики, управления тенантами и мониторинга системы.

## Структура

- `api/`: API маршруты для админ-панели.
- `services/`: Бизнес-логика (агрегация статистики, управление).

## Основные возможности

- Просмотр статистики по всем тенантам.
- Мониторинг активности пользователей.
- Управление системными настройками.
