# Analytics

## Обзор

Модуль аналитики для сбора, обработки и предоставления данных о работе системы и использовании AI-ассистентов.

## Структура

- `api/`: API для получения аналитических отчетов.
- `application/`: Прикладная логика обработки данных.
- `domain/`: Доменные модели аналитики.
- `infrastructure/`: Интеграция с хранилищами данных (ClickHouse/Postgres).

## Основные возможности

- Сбор метрик использования токенов.
- Анализ эффективности AI-ассистентов.
- Генерация отчетов для бизнеса.
