# Performance

## Обзор

Модуль для мониторинга и анализа производительности системы. Отвечает за сбор метрик, профилирование и выявление узких мест.

## Структура

- `api/`: API для получения отчетов о производительности.
- `domain/`: Модели метрик и логов производительности.
- `services/`: Логика анализа и агрегации данных.

## Основные возможности

- Сбор метрик времени выполнения запросов.
- Анализ потребления ресурсов.
- Профилирование кода.
