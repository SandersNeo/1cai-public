# Authentication & Authorization

## Обзор

Критически важный модуль, отвечающий за безопасность платформы. Реализует аутентификацию пользователей, управление сессиями и разграничение прав доступа (RBAC).

## Структура

- `api/`: Эндпоинты для входа, регистрации и обновления токенов.
- `application/`: Логика аутентификации и управления пользователями.
- `domain/`: Модели пользователей, ролей и токенов.
- `infrastructure/`: Работа с БД пользователей и JWT.

## Основные возможности

- JWT Authentication.
- Role-Based Access Control (RBAC).
- Управление жизненным циклом пользователей.
- Интеграция с внешними Identity Providers (опционально).
