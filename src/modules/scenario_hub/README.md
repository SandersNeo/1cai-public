# Scenario Hub

## Обзор

Централизованный репозиторий сценариев для AI-ассистентов. Управляет логикой выполнения сложных, многошаговых задач.

## Структура

- `nested_scenario_hub.py`: Основная логика вложенных сценариев.
- `api/`: API для управления сценариями.
- `domain/`: Модели сценариев и шагов.

## Основные возможности

- Хранение библиотеки сценариев.
- Версионирование логики.
- Динамическая загрузка сценариев.
