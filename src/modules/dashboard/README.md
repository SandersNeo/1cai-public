# Executive Dashboard

## Обзор

Модуль для предоставления высокоуровневых бизнес-метрик и KPI для руководителей и менеджеров.

## Структура

- `api/`: API для фронтенд-дашборда.
- `services/`: Расчет KPI, ROI и метрик здоровья системы.
- `domain/`: Модели виджетов и отчетов.
- `tasks/`: Фоновые задачи для обновления кэша метрик.

## Основные возможности

- Расчет ROI от внедрения AI.
- Мониторинг ключевых показателей эффективности.
- Визуализация трендов и прогнозов.
