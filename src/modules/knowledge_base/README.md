# Knowledge Base

## Обзор

Модуль базы знаний, обеспечивающий хранение, индексацию и семантический поиск информации (RAG - Retrieval Augmented Generation). Используется AI-ассистентами для получения контекста.

## Структура

- `api/`: Эндпоинты для поиска и управления документами.
- `services/`: Логика индексации и поиска (Vector Search).
- `domain/`: Модели документов и чанков.
- `infrastructure/`: Интеграция с векторными БД (Qdrant/Milvus).

## Основные возможности

- Загрузка и парсинг документов.
- Векторизация текста (Embeddings).
- Семантический поиск.
